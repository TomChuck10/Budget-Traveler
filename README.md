# 🌍 Budget Traveler

**Budget Traveler** to aplikacja webowa stworzona w ramach projektu zaliczeniowego na studia. Jej celem jest pomoc w planowaniu niskobudżetowych wyjazdów w oparciu o realne doświadczenia społeczności, a nie wyidealizowane materiały reklamowe.

---

## 🛠️ Technologie

- Frontend: **React + Vite**
- Stylowanie: **Tailwind CSS**
